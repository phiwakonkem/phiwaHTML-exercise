# HTML Beginner Exercise

A single-page HTML fundamentals exercise — practicing ordered/unordered lists, text formatting (bold, italics, centering), and email links. An early practice file from learning core HTML.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat-square&logo=html5&logoColor=white)

## Tech Stack

Plain HTML5 — no CSS, no JavaScript.

## Getting Started

```bash
git clone https://github.com/phiwakonkem/phiwaHTML-exercise.git
cd phiwaHTML-exercise
```

Open `index.html` directly in your browser.
